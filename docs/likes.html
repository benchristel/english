Lesson 3: Likes and Dislikes

To say "I like this", EnglishLANGNAME uses a paraphrase: AFIN#CAP maragood_0 yn reone#DEF mahere_0 drafor_1 vi1SG#LEN. The pattern for "PERSON likes THING" is AFIN#CAP maragood_0 THING^THING drafor_1 PERSON^PERSON. The PERSON is mutated.

Say who likes each thing. Examples:

  • Gandalf^Gandalf / reiodtravel_0#GER
    → AFIN#CAP maragood_0 reiodtravel_0#GER drafor_1 'andalf^Gandalf#LEN
  • Gimli^Gimli / ym monstone#DEF
    → AFIN#CAP maragood_0 ym monstone#DEF drafor_1 'imli^Gimli#LEN
  • Galadriel^Galadriel / ym mynnmorning#DEF | AFIN#CAP maragood_0 ym mynnmorning#DEF drafor_1 'aladriel^Galadriel#LEN
  • Bilbo^Bilbo / badodwalk_0#GER | AFIN#CAP maragood_0 badodwalk_0#GER drafor_1 Vilbo^Bilbo#LEN
  • Bilbo^Bilbo / yn gafytyafurnishing#DEF | AFIN#CAP maragood_0 yn gafytyafurnishing#DEF drafor_1 Vilbo^Bilbo#LEN
